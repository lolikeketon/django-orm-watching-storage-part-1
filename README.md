# Пульт охраны банка

Это внутренний проект для сотрудников банка **«Сияние»**.  
Если вы попали сюда случайно, запустить проект у вас не получится — у вас нет доступа к базе данных.  
Но вы можете посмотреть код, чтобы понять, как работает Django ORM без полноценного сервера.

## Что делает проект

**Пульт охраны** подключается к базе данных банка и выводит список всех пропусков, а также количество активных.  
Пример вывода:
```
Количество пропусков: 350
Активных пропусков: 27
```

## Установка

1. Склонируйте репозиторий и перейдите в папку проекта:
   ```bash
   git clone <ссылка_на_репозиторий>
   cd django-orm-watching-storage-part-1
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate   # для macOS / Linux
   venv\Scripts\activate      # для Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Создайте файл `.env` в корне проекта (если его нет) и добавьте в него данные для подключения:
   ```env
   HOST
   PORT
   USER
   PASSWORD
   SECRET_KEY
   ```

## Запуск

1. Убедитесь, что у вас есть доступ к базе данных Devman.  
2. Запустите скрипт:
   ```bash
   python main.py
   ```
3. В консоль выведется статистика по пропускам сотрудников.
